


	Projecto Books App

	El proyecto consiste en una aplicación, dónde un usuario puede crear su propia cuenta, y al mismo tiempo poderse loggear para ver su información
	de su cuenta en la libreria para ver el estado de sus libros.


	El proyecto consta en dos partes tanto cómo front end y back end.
        

	solamente en su carpeta principal, en la terminal hay que ejecutar los siguientes comandos para tenerlo listo y funcional:
	
			- npm run install:app    ---> instala las librerias de React cómo Node.
			- npm run start:app      ---> Inicia el servidor y la interfaz gráfica al mismo tiempo.


	
	Aquí dejaré el link de la documentación del servidor hecho en Node.Js Express:

			- LINK: https://documenter.getpostman.com/view/22441762/2s8YzMWjWG



	Maneja cómo base de datos PostgreSQL.
